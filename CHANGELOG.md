# 证据卫士 开发日志

## v0.2.0 (当前)

### 功能状态

| 模块 | 状态 | 说明 |
|------|:----:|------|
| 标题关键词检测（L1） | ✅ | 2秒轮询，60秒去重，排除照片查看器/资源管理器 |
| CDP 浏览器检测（L2） | ✅ | Chrome/Edge，需开调试端口，30秒轮询，不自动启动 |
| OCR 逐窗口检测（L3-IM） | ✅ | Tesseract，遍历IM窗口截图识别，30秒去重 |
| 手动取证 | ✅ | 截图+异步视频录制 |
| 关键词自动触发 | ✅ | 标题/CDP/OCR 命中→自动截图 |
| 三级采集模式 | ✅ | browser / desktop / full 可配置 |
| 三级通知模式 | ✅ | silent / toast / alert |
| Web 管理面板 | ✅ | http://127.0.0.1:58080 |
| 全局热键 | ✅ | 默认 Ctrl+Shift+F12，可配置 |
| DPAPI 加密 | ✅ | 默认开启 |
| 口令加密（跨机器） | ✅ | AES-256-GCM + PBKDF2 |
| 命令行解密工具 | ✅ | -decrypt 口令 -dir 目录 |
| 证据在线查看/导出 | ✅ | 单条导出 + 全部ZIP打包 |
| 浏览器调试快捷方式 | ✅ | 管理面板一键生成桌面快捷方式 |
| 系统组件状态检测 | ✅ | Tesseract/FFmpeg 可用性显示 |
| 开机自启 | ✅ | 托盘菜单开关 |
